Bem Vindo.

Este projeto faz parte de um dos conteúdos estudados e aplicados em sala no minicurso realizado em 19/10/2016 na Universidade Estadual do Tocantins. O Projeto visa levar uma compreensão estrutural do desenvolvimento de uma aplicação web utilizando os principais componentes adotados pela comunidade Java.

Introdução

O Mini-Curso de Desenvolvimento de Aplicações Web com Java e PrimeFaces tem o intuito apresentar os principais componentes que auxiliam no desenvolvimento de aplicações web tanto nos segmentos de softwares open-source quanto nos softwares comerciais. A fim de apresentar este curso de uma forma objetiva e didática, pretende-se ensinar alguns recursos de lógica de programação com Java 8 com utilização do Framework Web JSF 2.0, utilização de elementos gráficos web com a biblioteca PrimeFaces e para persistência de dados em um banco de dados local MySQL com a biblioteca JPA EclipseLink (ORM).

Desafio

Criar um projeto utilizando o Java EE (para web), importando bibliotecas a partir do gerenciador de dependências Maven, criando as classes de domínio (Modelos). O ambiente de desenvolvimento será o Intellij IDE de empresa JetBrains, com licença trial. O projeto será uma agenda de contatos, tal que, criarão manualmente um CRUD, com opções de inserção, edição, exclusão e listagem de informações referente a agenda desenvolvida. No final, os alunos compreenderão basicamente o funcionamento do Java Web e poderão desenvolver aplicações com utilização dos principais recursos Java do momento.
